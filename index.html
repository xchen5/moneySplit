Splitting Money for Stuff


Instructions:
1.) Using the text box, add names like the following example:
Person 1
Person 2
Person 3
2.) Press the Add Names button.
3.) Press Start Adding Items button.
4.) The browser should prompt you to input item prices. Input the price of item without the dollar sign. Press the OK button when done.
5.) Using the numbers assigned to each person, insert each person's number who is paying for said item.
6.) Repeat steps 4-5 and when there are no items left, do not input anything when it asks you for the price and press the OK button.
7.) The number of dollars each person should contribute will be on the side.
